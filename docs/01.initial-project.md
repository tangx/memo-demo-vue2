# 初始化项目

## 1. 安装 vue3 + ts 脚手架

1. 安装 `vue/cli`

```bash
npm i -g @vue/cli
npm install -g yarn  # vue create 需要
yarn config set registry https://registry.npm.taobao.org/ # yarn 配置国内源
```

2. 初始化项目

```bash
# 创建项目
vue create hmmemo

# 安装 ts 支持
cd hmmemo
vue add @vue/typescript

# 安装 vuex 支持
npm install vuex

```
